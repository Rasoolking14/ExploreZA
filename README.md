# 🌍 ExploreZA — Travel & Adventure Platform

## 📌 Project Name
**ExploreZA — Travel & Adventure Platform**

---

## 🚀 Startup Description
ExploreZA is a **non-profit initiative** that builds a digital platform to connect learners from under-resourced South African schools with outdoor adventures.  
The website showcases programmes, impact stories, donation options, and volunteer opportunities — all built with **HTML, CSS, and JavaScript**.

---

## ❗ Problem Statement
Most South African children from under-resourced schools never get the chance to explore their own country.  
A day in the mountains, a trip to the coast, or a wildlife reserve is simply out of reach. ExploreZA exists to close that gap by funding and running safe outdoor adventures at no cost to learners or schools.

## ✨ Features
- **Login System** with validation and error modal.  
- **Donation Page** with preset tiers and transparent impact reporting.  
- **Contact Page** with form validation and direct contact details.  
- **About & Programmes** section with mission, values, and trip sponsorship options.  
- **Responsive Navigation** with desktop/mobile menus and theme toggle.  

---

## 🧩 HTML Concepts Used
- Semantic structure (`<!DOCTYPE html>`, `<header>`, `<main>`, `<footer>`)  
- Forms with labels, inputs, and validation  
- Navigation menus (desktop & mobile)  
- Lists, tables, and structured content sections  
- Media embedding (`<img>`, `<svg>`)  

---

## 🎨 CSS Concepts Used
- External stylesheet (`styles.css`)  
- Utility classes inspired by Tailwind CSS  
- Responsive design with flexbox and grid  
- Hover effects and transitions  
- Dark/light theme styling with `data-theme`  

---

## ⚡ JavaScript Concepts Used
- DOM manipulation (`getElementById`, `querySelector`)  
- Event listeners for clicks and form submissions  
- Modal handling for login errors  
- Theme toggle with localStorage persistence  
- Smooth scrolling for anchor links
