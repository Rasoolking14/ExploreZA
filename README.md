# 🌍 ExploreZA

ExploreZA is a **non-profit web project** designed to give learners from under-resourced South African schools free access to outdoor adventures.  
This platform showcases programmes, impact stories, donation options, and volunteer opportunities — all built with **HTML, CSS, and JavaScript**.

---

## ✨ Features
- **Login System**
  - Secure entry with username/password validation.
  - Error modal for incorrect login attempts.
  - Dark mode enabled by default.

- **Donation Page**
  - Choose between one-time or monthly giving.
  - Preset donation tiers (R75, R150, R500, R2500).
  - Custom donation input field.
  - Impact transparency: shows exactly what each contribution funds.

- **Contact Page**
  - Contact form with validation (name, email, reason, message).
  - Direct contact details (email, phone, address).
  - Options to volunteer, nominate schools, donate gear, or partner corporately.

- **About & Programmes**
  - Mission statement and values (Access for All, Safety First, Transparency).
  - Programme cards with trip details (Table Mountain Hike, Aquarium visits, Acrobranch Adventure).
  - Sponsorship options for learners and schools.

- **Responsive Navigation**
  - Desktop and mobile menus.
  - Theme toggle (light/dark mode).
  - Sign-in, donate, and volunteer actions.
